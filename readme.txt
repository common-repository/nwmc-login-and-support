== Description ==
This plugin will add the nwmc logo to the bottom of your login box and NW Media Collective's support info to the bottom of your admin area in WordPress.

For users that have signed up for one of our <a href="https://northwestmediacollective.com/website-maintenance/" target="_blank">maintenance plans</a>


== Installation ==
- Upload or install nwmc-login-support.zip
- Activate the plugin
